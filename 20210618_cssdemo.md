*우선 순위가 같을때는 마지막에 정의된다*

```css
p{
	  color: red;
	  font-size: 30px;
	}
p{
	  background-color: yellow;
	   color: blue; /*우선순위가 같을때 마지막에 정의된 스타일이 적용됨 */
  }



/* 0002 = 0001 + 0001 */
 body h1 {color: red;}
 
 /* 0001 */
 h1{color: orange;}

/* 각각 0001 */
 h1,p{color: orchid;}

>>>h1은 컬러 색깔: red

/* h1요소에서 blue라는 클래스를 가질때 */
/* 0011 = 0001 + 0010 */
h1.blue{color: aqua;}



/* p요소에서 속성중 class속성에 unit값이 설정된 요소에
 스타일을 적용한다 */
 p.unit {
        background-color: goldenrod;
 }

/* p요소에서 자손속성중 class속성에 unit값이 설정된 요소에
 스타일을 적용한다 */
 p .unit{
	 background-color: greenyellow; 
 }

>>> 띄어쓰기에 따라 적용 범위가 달라진다



/* > 자식을 의미한다 */
div> span{
	color: greenyellow;
}

/* 공백은 자손 선택지이다 */
div font {
	background-color: indigo;
}
/* +인접은(형제) 선택지이다 */
div + p{
	color: red;
}

/* [속성명='값'] 속성 선택자이다 */
p[class='one']{
	border-style: solid;
	border-width: 1px;
	border-color: orange;
}

/* ,(쉼표)는 그룹화 선택자이다 */
img, address{
	border-style: solid;
	border-width: 1px;
	border-color: aqua;
}



/* pesudo class을 작성할때 link -> :visited -> hover -> :active 순으로 작성한다 */

/* 한번도 방문하지 않은 상태의 스타일이다 */
a:link{
	color: red;
}
/* 방문했던 링크의 스타일이다 */
a:visited{
	color: royalblue;
}

/* 마우스 포인터를 요소에 올려올때의 스타일을 지정한다 */
a:hover{
	color: salmon;
}

/* 요소가 활성 상태일때 스타일을 지정한다 */
a:active{
	color: skyblue;
}




/* p요소가 부모 요소의 첫번째 자식요소로 사용될때 스타일 적용 */
  p:first-child{
	  background-color: chartreuse;
  }

/* 2번째 자식요소는 스타일 적용됨 */
p:nth-child(2){
	background-color: red;
}



p:nth-child(even){
	background-color: red;
}

p:nth-of-type(even){
	color: blue;
}

<p>computer</p>
<span>test</span>
<p>monitor</p>
<p>television</p>
<p>desk</p>
<p>top</p>

>>>같은 짝수자리이지만 p:nth-child(even)은 <span>이 있기때문에 monitor가 1로 시작되고
p:nth-of-type(even)은 cmmputer가 1로 시작되고 test는 건너뛰고 monitor가 2번으로 된다



CSS박스모델은 Html요소를 둘러싸고 있는  박스에 대한 정의이며 하나의 요소에 대한 박스는 콘텐츠(내용), 패딩(안쪽여백),보더(테두리),
마진(바깥쪽여백)으로 구성된다.
```
